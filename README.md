

# 📊 AgroBank: SQL para Análise de Dados e Crédito (Fintech)
Este repositório contém uma série de consultas SQL desenvolvidas para o **AgroBank**, uma fintech focada em gestão financeira e concessão de crédito para clientes PF e PME. O objetivo deste projeto é demonstrar proficiência em SQL para resolver problemas de negócio, análise de risco, compliance e performance comercial.

## 📂 Estrutura das Análises
As consultas estão organizadas em níveis de complexidade, abordando:

1. **Visão de Negócio:** Volume de crédito por produto, perfil de renda e performance de gerentes.
2. **Gestão de Risco & Crédito:** Cálculo de comprometimento de renda, análise de inadimplência (Time-to-Default) e indicadores de atraso.
3. **Análise de Dados Avançada:** * **Análise de Safra (Cohort):** Crescimento mensal de volume concedido.
* **Curva ABC (Pareto):** Identificação de clientes que concentram 80% da receita.
* **Gaps & Islands:** Identificação de uso consecutivo da conta.
4. **Compliance e Prevenção à Fraude:** Identificação de contas inativas com faturas abertas e picos de consumo atípicos.
5. **Engajamento (RFM):** Classificação de clientes por recência e frequência de transações.
